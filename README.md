# Decimal Support

Decimal 是一款面向 macOS 的原生文件管理应用，基于 Johnny Decimal 方法帮助你建立清晰、可维护的文件组织结构。

## Support

如果你在使用 Decimal 时遇到问题，可以通过以下方式获取帮助：

- 提交 Issue：请在本仓库的 Issues 页面描述问题、复现步骤和 macOS 版本
- 功能建议：欢迎通过 Issues 提交改进建议
- App Store 用户：也可以通过本页面作为 Support URL 访问支持信息

## Recommended issue template

提交问题时，建议附带以下信息：

- Decimal 版本
- macOS 版本
- 问题发生时的操作步骤
- 期望行为
- 实际行为
- 如可提供，请附截图或错误日志

## About Decimal

Decimal 提供以下核心能力：

- 基于 Johnny Decimal 的文件结构管理
- 本地优先的 Vault 浏览与整理
- AI 辅助文件归类
- 智能搜索与快速定位
- JDex 编辑与维护
- 与日历相关的工作流支持

## Notes

- Decimal 的核心体验以本地优先为原则
- 启用 AI 功能时，仅会按功能需要将必要元数据发送给你选择的第三方 AI 服务商
- 具体隐私说明以隐私政策页面为准
